# Password Generator (REACT)
A modern and responsive Password Generator built using React and Tailwind CSS.
This application allows users to generate strong random passwords with customizable options such as password length, numbers, and special characters.

# Features
1. Generate secure random passwords
2. Adjustable password length
3. Include/exclude:
   Numbers
   Special characters
4. One-click copy to clipboard
5. Responsive UI
6. Built with React Hooks:
useState
useEffect
useCallback
useRef
# Tech Stack
React
Tailwind CSS
JavaScript (ES6+)
Vite

# Installation & Setup

Clone the repository:

git clone https://github.com/your-username/password-generator.git

Navigate to the project folder:

cd password-generator

Install dependencies:

npm install

Run the development server:

npm run dev
